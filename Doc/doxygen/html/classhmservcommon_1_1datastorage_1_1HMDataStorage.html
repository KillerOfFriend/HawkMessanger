Подробное описание

The HMDataStorage class - Интерфейс, описывающий хранилище данных сервера

Хранит информацию о: 1) Пользователях системы. 2) Перечень контактов пользователей системы. 3) Группах системы. 4) Перечень участников групп системы. 5) Переписку групп системы.
